# HMS Client Website

Standalone guest/client frontend.

- Dev URL: `http://localhost:8080`
- Login URL: `http://localhost:8080/client-login`
- Guest app URL: `http://localhost:8080/guest`
- Shared backend API: `http://localhost:8787/api`

Run manually:

```powershell
cd "C:\Users\ACXIOM\Desktop\Acxiom Files\hms\client"
npm install
npm run dev
```

The staff/admin frontend is separate under `../staff` and runs on port `8081`.
